========================
Pmccabe for Sublime Text
========================

This package provides easy access to 'pmccabe' code complexity analysis for C/C++.

Requirements
------------

- pmccabe_ installed and working

.. _pmccabe: https://people.debian.org/~bame/pmccabe/download.html
